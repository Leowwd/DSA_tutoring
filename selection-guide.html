Data Structure Selection Guide

This page answers the question students ask most often: "How do I know which structure to use?" Start from the operation, then choose the structure.

Common pairwise decisions

HashMap vs TreeMap

Use HashMap when order does not matter and expected O(1) lookup is enough. Use TreeMap when sorted keys, range views, floor/ceiling, or ordered iteration are part of the answer.

PriorityQueue vs sorting

Sort when you need the full order. Use a priority queue when you repeatedly need only the current best item or only the top k items.

ArrayList vs LinkedList

In Java, ArrayList is usually the default list because indexing and iteration are efficient. LinkedList is rarely the first choice for DSA practice; for stack/queue/deque behavior, ArrayDeque is normally cleaner.

HashSet vs boolean array

Use a boolean array when keys are small nonnegative integers with a known range. Use HashSet when keys are sparse, large, strings, objects, or unknown in advance.
