DSA Debugging Checklist

Most DSA bugs are not mysterious. They come from boundaries, missing base cases, wrong invariants, or using the wrong structure for the operation.

Universal edge cases

Input typeAlways test
Array / ListEmpty, one item, duplicates, already sorted, reversed, negative values.
StringEmpty, one character, repeated characters, spaces, case sensitivity.
Linked listEmpty head, one node, removing head/tail, cycle if relevant.
TreeNull root, one node, skewed tree, duplicate policy, invalid BST.
GraphDisconnected graph, cycle, self-loop, isolated vertex, directed vs undirected.
DPSmallest input, impossible state, repeated state, overflow in counting.

Bug patterns by topic

TopicLikely bugDebug question
Binary searchInfinite loop or missed answerDoes the range shrink every iteration?
Two pointersWrong pointer movesWhich move preserves the invariant?
Sliding windowWindow not valid when answer updatesDo I update answer before or after shrinking?
RecursionStack overflowIs there a base case and progress toward it?
BacktrackingMissing or duplicate answersDid I undo every choice?
HeapWrong priority removedIs the comparator defining min-first or max-first?
HashMapWrong countsWhen do I increment, decrement, or remove keys?
GraphRepeated visits or missed nodesWhen do I mark a node as seen?
DPWrong table valuesDoes every transition match the state definition?

Trace before changing code

DEBUGGING ROUTINE:
    1. Pick the smallest failing input.
    2. Write the expected output by hand.
    3. Trace variables after every loop or recursive call.
    4. Compare the trace to the invariant.
    5. Fix the first line where the invariant breaks.

Java-specific mistakes

  • Using == for string equality instead of equals.
  • Writing a comparator as subtraction and causing integer overflow.
  • Modifying a collection while iterating with a for-each loop.
  • Forgetting that arrays are mutable and passed by reference value.
  • Assuming HashMap iteration order is stable.
  • Forgetting generic type parameters and falling back to raw types.
